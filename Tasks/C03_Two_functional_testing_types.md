# C03 - Two Functional Testing Types

## 1. Smoke Testing

### Definition
Smoke Testing verifies that the most important functionalities of the application work correctly after a new build or deployment.

### Application
I opened the "Asociația Oportunități și Cariere" website and verified that:
- the homepage loads successfully;
- the navigation menu is displayed;
- the main pages open correctly without errors.

### Why I chose it
I chose Smoke Testing because it quickly verifies that the website's core functionalities are working properly.

---

## 2. Positive Testing

### Definition
Positive Testing verifies that the application behaves correctly when the user performs valid actions and uses valid input.

### Application
I navigated through the website using the menu and verified that all pages opened correctly and displayed the expected information.

### Why I chose it
I chose Positive Testing because it confirms that the website works correctly under normal user conditions.
